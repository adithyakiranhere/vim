EXERCISE 4: SAVING AND QUITTING
---------------------------------
Practice these commands (don't worry, you won't lose this file):

1. Type :w and press Enter        -- this saves the file
2. Type :q and press Enter        -- this quits (will fail if you edited)
3. Type :wq and press Enter       -- this saves and quits
4. Type :q! and press Enter       -- this quits WITHOUT saving
5. Try ZZ in normal mode          -- shortcut for :wq
6. Try ZQ in normal mode          -- shortcut for :q!

Tip: After quitting, reopen the file with:  nvim session1_practice.txt


EXERCISE 5: MODE AWARENESS
-----------------------------
At any time you can check which mode you're in by looking at
the bottom-left of your Neovim screen. It will show:
  -- INSERT --    when in Insert mode
  -- VISUAL --    when in Visual mode
  (nothing)       when in Normal mode

Practice switching modes rapidly:
  i  -> Esc -> a -> Esc -> o -> Esc -> I -> Esc -> A -> Esc -> O -> Esc

Do this 10 times until it becomes muscle memory!


EXERCISE 6: THE GOLDEN RULE
------------------------------
ALWAYS return to NORMAL mode when you're done typing.

Normal mode is your HOME. You visit Insert mode to type,
then come back home. Think of it like this:

  Normal mode = Standing (ready to act)
  Insert mode = Sitting down to write
  Visual mode = Highlighting with a marker
  Command mode = Talking to the editor

You should spend 80% of your time in Normal mode.
